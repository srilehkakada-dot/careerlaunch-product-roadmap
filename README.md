# 🗺️ CareerLaunch Product Roadmap

## 📌 Overview

This project presents a product roadmap for **CareerLaunch**, a job search platform designed for students, fresh graduates, and career switchers in India.

The roadmap was created using the **RICE prioritization framework** to evaluate features based on Reach, Impact, Confidence, and Effort.

---

## 🎯 Objectives

- Prioritize product features
- Apply the RICE framework
- Create a roadmap
- Build a visual prototype
- Present product strategy
